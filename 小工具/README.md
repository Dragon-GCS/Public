# 小工具们
主要存放一些平时工作中比较有用的小脚本

* Cell_IO.py

    细胞库管理程序，用于细胞出入库和库存查询。
    
    cell_info.json为python字典保存的json文件。
    
        key: str ,保存位置  例A1-1-5
        value: list,[cell_name, other_infomations]
  
* PNG2JPG

    将文件夹中所有PNG转换为JPG，主要用于腾出存储空间

* alloy_content_calculator

    计算合金中各个组分的含量（四元一次方程组计算）
